Thank you for your interest in contributing! This project is a modern React + Vite + Tailwind CSS portfolio. We welcome improvements, bug fixes, and new features.

## How to Contribute

1. **Fork the repository** and clone your fork.
2. **Create a new branch** for your feature or fix:
   ```sh
   git checkout -b feature/your-feature-name
   ```
3. **Make your changes** following best practices:
   - Use modular, reusable components in `src/components/`
   - Keep pages in `src/pages/`
   - Use Tailwind CSS for styling
   - Add animations with framer-motion where appropriate
   - Ensure mobile responsiveness and accessibility
4. **Test your changes** locally:
   ```sh
   npm run dev
   ```
5. **Commit and push** your changes:
   ```sh
   git commit -m "Describe your change"
   git push origin feature/your-feature-name
   ```
6. **Open a Pull Request** on GitHub and describe your changes.

## Code Style
- Use functional React components and hooks
- Prefer ES6+ syntax
- Keep code clean and maintainable
- Write clear commit messages

## Reporting Issues
If you find a bug or have a feature request, please open an issue on GitHub with details and steps to reproduce.

## Contact
For questions, reach out via GitHub Issues.
